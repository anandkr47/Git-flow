# Code Flowchart Generator Chrome Extension

A Chrome extension that automatically generates interactive flowcharts from code blocks in GitHub repositories.

## Features

- Dynamic flowchart generation from code blocks on GitHub pages
- Support for multiple programming languages (JavaScript, TypeScript, Python, Java)
- Interactive flowchart nodes with code navigation
- Customizable themes (Light, Dark, GitHub) and detail levels (Low, Medium, High)
- Automatic flowchart generation with toggle option
- Built with TypeScript, D3.js, and Mermaid.js

## Usage

1. **Install and Load the Extension**: Follow the installation steps above to load the extension in Chrome.

2. **Navigate to a GitHub Repository**: Open any GitHub repository page that contains code files.

3. **View Code Blocks**: The extension automatically detects code blocks on GitHub pages (e.g., in README files, issues, or pull requests).

4. **Generate Flowcharts**: 
   - If auto-generation is enabled (default), flowcharts will appear automatically below supported code blocks.
   - Click on flowchart nodes to navigate to the corresponding code lines.
   - Hover over nodes for additional information.

5. **Customize Settings**:
   - Click the extension icon in the Chrome toolbar to open the popup.
   - Adjust theme (Light/Dark/GitHub), detail level (Low/Medium/High), and toggle auto-generation.
   - Settings are saved automatically and apply to new tabs.

6. **Supported Languages**: The extension analyzes code in JavaScript, TypeScript, Python, and Java files, identifying functions, conditionals, loops, and statements to build flowcharts.

## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the extension:
   ```bash
   npm run build
   ```
4. Load the extension in Chrome:
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the `dist` directory of this project

## Development

- `npm run watch` - Watch for changes and rebuild automatically
- `npm test` - Run Jest tests with coverage reporting

## Project Structure

- `src/` - TypeScript source files
  - `content/content.ts` - Content script for GitHub page integration and code block detection
  - `background/background.ts` - Extension background service worker
  - `ui/popup.ts` - Extension popup UI logic for settings
  - `core/codeAnalyzer.ts` - Code analysis and parsing logic
  - `core/flowchartRenderer.ts` - D3.js-based flowchart rendering
  - `types/index.ts` - TypeScript type definitions
  - `__tests__/` - Jest test files
- `public/` - Static files
  - `popup.html` - Extension popup UI
- `src/ui/styles/` - CSS styles
  - `flowchart.css` - Flowchart styling
- `dist/` - Built extension files (generated by webpack)
- `coverage/` - Test coverage reports (generated by Jest)

## Technologies Used

- **TypeScript** - Type-safe JavaScript
- **D3.js** - Data visualization library for SVG flowcharts
- **Mermaid.js** - Diagram generation (dependency included)
- **Webpack** - Module bundler and build tool
- **Jest** - Testing framework
- **Chrome Extension Manifest V3** - Modern extension API
